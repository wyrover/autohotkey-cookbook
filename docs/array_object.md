## Associative Arrays

```js
; Create a new array
LastNames := {}

; Add some names
LastNames.Insert("John", "Hancock")
LastNames.Insert("Tom", "Hanks")
LastNames.Insert("Will", "Smith")

; Get Tom's last name
MsgBox % LastNames["Tom"]
MsgBox % LastNames.Tom
```
